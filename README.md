# One Page

View and edit all your daily notes in one continuous, scrollable page.

![One Page Demo](https://raw.githubusercontent.com/Quorafind/Obsidian-Daily-Notes-Editor/master/image/Daily-Note-View.gif)

## Features

- **Continuous editing** - Edit multiple daily notes in a single scrollable view
- **Time filters** - Filter by week, month, quarter, year, or custom date range
- **Multiple modes** - View daily notes, folder contents, or notes by tag
- **Sort options** - Sort by creation or modification time
- **Presets** - Save custom views for quick access

## Installation

Install from Obsidian Community Plugins (search "One Page")

## Usage

1. Click the calendar icon in the left ribbon or run command "Open One Page"
2. Use the toolbar to filter by time range, mode, or sort order
3. Click any note title to open it in a separate pane

## Settings

- Hide frontmatter
- Hide backlinks
- Open on startup
